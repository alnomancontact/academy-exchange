About Academy Exchange

A student-driven platform built to make academic resources more accessible, affordable, and collaborative.

🎯

Project Purpose

Designed to solve the common problem of accessing affordable academic resources. Students can list used books and study materials for sale or exchange, helping peers save money and share knowledge.

💡

Key Features

  • Buy/sell/exchange books and study materials
  • Searchable listings categorized by course/semester
  • Secure login & registration system
  • Responsive design with Tailwind CSS
👨‍💻

Development Overview

Built by a team of undergraduate students applying core software engineering principles. It combines a modern frontend (HTML, Tailwind, JS) with a powerful Django backend.
